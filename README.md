
# AI Math Tutor Agent 🤖

This project is built using Google ADK.

## 💡 Features
- Solves algebra problems step by step
- Guides students instead of giving direct answers
- Friendly tutor behavior

## 🛠️ Tech Used
- Google ADK
- Gemini model

## 🚀 How to Run

1. Add your API key in `.env`
2. Run:
   adk web my_config_agent

## 📌 Example

Input:
What is 2x + 5 = 13?

Output:
Step-by-step explanation
